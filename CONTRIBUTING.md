# Contributing

We welcome contributions to improve this framework.

## How to Contribute

1. **Open an issue** — describe the improvement or suggestion
2. **Fork the repo** — make changes on a feature branch
3. **Submit a pull request** — reference the issue

## Guidelines

- Keep questionnaires focused and actionable
- Scoring rubrics must be objective and repeatable
- Templates should be platform-specific to IBM Cloud and watsonx
- All contributions must be compatible with the Apache 2.0 license

## Questions?

Open an issue or reach out at [proximaintel.com/contact](https://proximaintel.com/contact).
