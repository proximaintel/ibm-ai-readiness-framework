# Engagement Guide

## How to Run the AI Readiness Assessment

### Pre-Engagement
- [ ] Confirm scope and stakeholder list
- [ ] Share questionnaires in advance (optional — some clients prefer live)
- [ ] Schedule interviews (5-8 sessions across 2 weeks)
- [ ] Request access to IBM Cloud console (read-only)
- [ ] NDA and engagement letter signed

### Week 1: Discovery
- [ ] Day 1: Kickoff — introductions, scope confirmation, logistics
- [ ] Day 2-3: Infrastructure and data interviews
- [ ] Day 4: AI initiatives deep-dive and use case workshop
- [ ] Day 5: Security, compliance, and organizational readiness

### Week 2: Analysis & Delivery
- [ ] Day 6-7: Score all dimensions, identify gaps
- [ ] Day 8: Build roadmap, prioritize use cases
- [ ] Day 9: Write report, prepare executive briefing
- [ ] Day 10: Deliver executive briefing, Q&A, next steps

### Post-Engagement
- [ ] Deliver final report (PDF)
- [ ] Share roadmap and gap analysis matrix
- [ ] Schedule follow-up call (2 weeks out)
- [ ] Propose next engagement (if appropriate)
